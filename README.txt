O Mundo de Wumpus é um ambiente de grade onde há um monstro chamado Wumpus que devora quem chegar na sua casa. Em algumas casas existem poços que engolem quem pisar nelas. Em uma das casas há ouro, o qual o guerreiro precisa pegar, voltar para a casa inicial e sair da caverna.
Nas casas adjacentes ao Wumpus (não nas diagonais) o guerreiro perceberá fedor.
Nas casas adjacentes aos poços (não nas diagonais), o guerreiro perceberá uma brisa.
Na casa onde está o ouro, o guerreiro perceberá o ouro.
O guerreiro está munido de uma única flecha, que poderá ser disparada para matar o Wumpus.
Para acertar o Wumpus, o guerreiro precisa se encontrar virado de frente para o monstro, não importando a distância de casas.
Quando morto, o Wumpus solta um grito que pode ser escutado de qualquer lugar da caverna.
Para cada ação executada pelo querreiro, subtrai-se um ponto.
Para o ouro pego, soma-se 100 pontos.
Caso caia em um poço ou seja devorado pelo Wumpus subtrai-se 1000 pontos.

As ações permitidas são:
virar para a direita (Ctrl direito);
virar para esquerda (Ctrl esquerdo);
andar para frente (setas);
pegar o ouro (A);
atirar flecha (Q).

O guerreiro só anda para frente, portanto para virar para uma a direção desejada precisa estar virado de frente para a aquela direção.
